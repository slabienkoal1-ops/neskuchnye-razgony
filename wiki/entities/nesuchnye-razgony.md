## Технический процесс регистрации

### Форма на лендинге (Tilda)
- Поля данных участника (минимум: email, имя, телефон)
- Отработка цели Яндекс.Метрики при отправке формы

### После отправки формы

```
Форма (Tilda)
├── → Страница благодарности — выбор мессенджера для подписки
├── → Sendsay — отбивка письма-подтверждения (email)
├── → Bitrix24 — создание сделки по ТЗ (воронка: Бесплатные мероприятия)
└── → Google Таблицы — сбор данных для аналитики
```

### Страница благодарности
Основной способ выбора мессенджера для подписки. После отправки формы участник попадает на страницу благодарности, где выбирает удобный мессенджер (кнопки-ссылки).
